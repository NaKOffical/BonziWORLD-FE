Here some tips to make a double rocket fludd.

1. Test any level if you want to do a rocket fludd, but please place the item to test it.

2. Press C and press pause rapidly when rocket starts flushing.

3. Before does make a animation, press C again.

Almost any level in full game can get more secrets.
